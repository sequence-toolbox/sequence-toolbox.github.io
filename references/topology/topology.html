Topology¶

Definition of the Topology class.

This module provides a definition of the Topology class, which can be used to manage a network’s structure. Topology instances automatically perform many useful network functions.

class src.topology.topology.Topology(name: str, timeline: Timeline)¶

Class for managing network topologies.

The topology class provies a simple interface for managing the nodes and connections in a network. A network may also be generated using an external json file.

name¶

label for topology.

Type

str

timeline¶

timeline to be used for all objects in network.

Type

Timeline

nodes¶

mapping of node names to node objects.

Type

Dict[str, Node]

qchannels¶

list of quantum channel objects in network.

Type

List[QuantumChannel]

cchannels¶

list of classical channel objects in network.

Type

List[ClassicalChannel]

graph¶

(Dict[str, Dict[str, float]]): internal representation of quantum graph.

generate_forwarding_table(starting_node: str) → dict¶

Method to create forwarding table for static routing protocol.

Generates a mapping of destination nodes to next node for routing using Dijkstra’s algorithm.

Parameters

node (str) – name of node for which to generate table.

load_config(config_file: str) → None¶

Method to load a network configuration file.

Network should be specified in json format. Will populate nodes, qchannels, cchannels, and graph fields. Will also generate and install forwarding tables for quantum router nodes.

Parameters

config_file (str) – path to json file specifying network.

Side Effects:

Will modify graph, graph_no_middle, qchannels, and cchannels attributes.
